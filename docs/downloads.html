There are installation packages available for Linux and Windows:

  • Windows
  • Debian/Ubuntu

Both installers will configure the server to start automatically when your computer boots. There is also a separate zip package that makes no changes to your system configuration, and will not automatically boot the server:

  • Non-Installation

This version works on any operating system, so long as Java is installed. To run it, extract the package contents and double-click on Server.jar. If any firewall notices appear, click Allow.

Troubleshooting

  • If you experience the mouse cursor disappearing while using the non-installation version on Ubuntu, consider switching to the installer package. It contains a script that fixes the problem during installation. Alternatively, you can fix it yourself using the method described in this thread.
